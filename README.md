# עוזר AI קולי בטלפון - חיבור ימות המשיח ל-Claude

מדריך זה ילווה אותך שלב-אחר-שלב בהקמת מערכת שמאפשרת לכל אחד להתקשר למספר טלפון רגיל (גם מטלפון כשר, בלי אינטרנט בטלפון עצמו) ולנהל שיחה קולית חופשית עם בינה מלאכותית, בעברית.

**חשוב לדעת מראש:**
- הטלפון של המתקשר לא צריך אינטרנט בכלל - הוא רק מחייג שיחה רגילה.
- המערכת דורשת שרת קטן שרץ באינטרנט (נבנה כבר בשבילך, בקבצים שבתיקייה הזו) ומפתח API בתשלום לפי שימוש של Anthropic (עלות נמוכה מאוד לשיחה, בדרך כלל כמה סנטים).
- זהו פרויקט "עשה זאת בעצמך" מבוסס תיעוד רשמי של ימות המשיח - יכול להיות שיידרשו כמה כיוונונים קטנים אחרי הבדיקה הראשונה בשטח, וזה נורמלי לגמרי.

---

## שלב 1: השגת מפתח API של Anthropic (Claude)

1. היכנס ל-https://console.anthropic.com והרשם / התחבר.
2. בתפריט הצד, מצא **API Keys** ולחץ **Create Key**.
3. העתק את המפתח (מתחיל ב-`sk-ant-...`) ושמור אותו במקום בטוח - הוא יידרש בהמשך. **הוא לא יוצג שוב באתר.**
4. תחת **Billing**, טען כרטיס אשראי והוסף קרדיט (אפשר להתחיל עם סכום קטן, למשל 5$-10$ - זה מספיק להמון שיחות בהתחלה).

---

## שלב 2: העלאת הקוד ל-GitHub

(GitHub הוא אתר חינמי לאחסון קוד, שממנו נעלה את הפרויקט לשרת בשלב הבא)

1. היכנס ל-https://github.com והרשם / התחבר.
2. לחץ על **+** למעלה ואז **New repository**.
3. תן שם, למשל `yemot-ai-phone`, השאר אותו **Private** (פרטי) אם תרצה, ולחץ **Create repository**.
4. בעמוד שנפתח, לחץ על הקישור **uploading an existing file**.
5. גרור לתוך הדפדפן את **כל הקבצים** שבתיקייה הזו (`server.js`, `package.json`, `.env.example`, `.gitignore`, `README.md`) - **חוץ מ-`.env`** אם יצרת כזה בהמשך (אין ליצור אותו כאן בכלל).
6. לחץ **Commit changes** בתחתית העמוד.

---

## שלב 3: הרצת השרת ב-Render.com (אחסון חינמי)

1. היכנס ל-https://render.com והרשם (אפשר להירשם עם חשבון ה-GitHub שיצרת).
2. בלוח הבקרה, לחץ **New** ← **Web Service**.
3. חבר את חשבון ה-GitHub שלך ובחר את המאגר (`repository`) `yemot-ai-phone` שיצרת.
4. במסך ההגדרות:
   - **Name**: שם חופשי, למשל `yemot-ai-phone`
   - **Region**: כל אזור (מומלץ הקרוב ביותר, למשל פרנקפורט)
   - **Branch**: `main`
   - **Runtime**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: `Free` (חינמי) - לכתחילה מספיק כדי לבדוק שהכל עובד
5. גלול למטה ל-**Environment Variables** ולחץ **Add Environment Variable**:
   - `ANTHROPIC_API_KEY` = המפתח שהעתקת בשלב 1
   - (אופציונלי) `CLAUDE_MODEL` = `claude-sonnet-5`
6. לחץ **Create Web Service**. הפריסה תיקח כ-2-3 דקות.
7. כשהסטטוס יהיה **Live**, תראה למעלה כתובת מהצורה:
   `https://yemot-ai-phone.onrender.com`
   **העתק כתובת זו** - נזדקק לה בשלב הבא.

**הערה חשובה על התוכנית החינמית:** בתוכנית ה-Free, השרת "נרדם" אחרי כ-15 דקות של חוסר פעילות, וההתעוררות בשיחה הראשונה לאחר שינה לוקחת כ-30-50 שניות (המתקשר ישמע שקט/צלצול ממושך בהתחלה). אם זה מפריע, ניתן לשדרג בהמשך לתוכנית בתשלום (כ-7$ לחודש) שלא נרדמת.

---

## שלב 4: הגדרת שלוחת API בימות המשיח

1. היכנס לממשק הניהול של ימות המשיח שלך.
2. עבור להגדרות הקו/שלוחות, וצור **שלוחה חדשה מסוג "שלוחת API"**.
3. בהגדרות השלוחה, מצא את השדה של **כתובת ה-URL** (לפעמים נקרא `api_url` או דומה) והדבק בו את הכתובת מ-Render, בתוספת סלאש בסוף:
   `https://yemot-ai-phone.onrender.com/`
4. ודא שהשיטה מוגדרת ל-**POST** (בימות: `api_url_post=yes`) - כך מומלץ לעבודה עם השרת שבנינו.
5. ודא ששפת ברירת המחדל של השלוחה (`lang`) מוגדרת לעברית (`he`), אם יש שדה כזה.
6. שמור את ההגדרות.
7. חבר את השלוחה הזו למספר הטלפון הרלוונטי (כניסה ראשית של הקו, או שלוחה בתפריט קיים).

---

## שלב 5: בדיקה

1. התקשר למספר הטלפון שלך (מכל טלפון, כולל טלפון כשר).
2. אתה אמור לשמוע: **"שלום! אני עוזר קולי חכם..."**
3. דבר בעברית רגילה - שאל שאלה, וחכה לתשובה.
4. אחרי כ-3 שניות שקט מהצד שלך, המערכת תשלח את מה שאמרת ל-Claude ותקריא לך תשובה, ואז תמשיך להאזין לשאלה הבאה.
5. כדי לסיים שיחה, אפשר לומר "תודה" או "להתראות", או פשוט לנתק.

אם משהו לא עובד, ראה את פרק **פתרון בעיות** למטה.

---

## איך לשנות את "האישיות" של העוזר

בקובץ `server.js`, חפש את המשתנה `SYSTEM_PROMPT` בתחילת הקובץ - זה הטקסט שמגדיר איך ה-AI מתנהג ומדבר. אפשר לערוך אותו חופשי (למשל להוסיף תחום התמחות, טון מסוים, וכו'), לשמור, ולהעלות מחדש ל-GitHub - Render יעדכן את השרת אוטומטית תוך דקה-שתיים.

גם את הודעת הפתיחה אפשר לשנות במשתנה `GREETING`.

## פתרון בעיות נפוצות

- **השיחה מתנתקת מיד / לא נשמע כלום**: בדוק ב-Render, בלשונית **Logs**, אם יש שגיאות. לרוב זו בעיה במפתח ה-API (חסר, שגוי, או אין קרדיט בחשבון Anthropic).
- **יש עיכוב ארוך בתחילת השיחה**: זה כנראה "התעוררות" השרת מהתוכנית החינמית (ראה שלב 3) - זה נורמלי, או שדרג לתוכנית בתשלום.
- **המערכת לא מזהה נכון את מה שאני אומר**: ודא שהגדרת בשלוחה שפה עברית, ונסה לדבר לאט וברור. אפשר גם לכוונן את הערך `quiet_max` בקובץ `server.js` (למשל להגדיל ל-4 או 5 שניות אם המערכת "חותכת" אותך מוקדם מדי).
- **תשובות ה-AI נשמעות מוזר / חסרות מילים**: ימות המשיח לא יכול להקריא תווים מסוימים (נקודה, מקף, גרש, גרשיים, `&`) - הקוד כבר מסנן אותם אוטומטית, אבל אם רואים בעיה נוספת אפשר לעדכן את הפונקציה `cleanText` בקובץ `server.js`.
- **רוצה קול טבעי יותר (לא הקראה רובוטית)**: הגרסה הנוכחית משתמשת במנוע ההקראה המובנה של ימות המשיח (חינמי אבל נשמע ממוחשב). בהמשך אפשר לשדרג לקול טבעי (כמו Microsoft Edge TTS או Google TTS) על ידי יצירת קובץ קול ושליחתו לימות כקובץ הקלטה - זה שינוי גדול יותר בקוד, ואשמח לעזור לבנות את זה כשתרצה.

---

## עלויות משוערות

- **Anthropic (Claude)**: כמה סנטים בודדים לשיחה טיפוסית (תלוי באורך השיחה).
- **Render.com**: חינמי בתוכנית הבסיסית (עם מגבלת "שינה"), או כ-7$ לחודש לפעילות רציפה בלי עיכובים.
- **ימות המשיח**: לפי המנוי הקיים שלך - אין עלות נוספת מעבר לזה עבור השימוש בשלוחת ה-API עצמה.
